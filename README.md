## Using BETO RoBERTa and RNN for humor detection in social media data.

The PW3_S5 script has the code implemented in order to fine tune BETO for binary classification, regression and multilabel classification tasks.

The roberta script has the code implemented for fine tune RoBERTa for binary classfication.

RNN scrip is a first approach of a Recurrent network in order to classify the different tweets of the dataset. This model is giving the worst results so far.
